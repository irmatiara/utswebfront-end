Sejarah HTML

HTML pertama kali diciptakan oleh Tim Berners-Lee pada tahun 1991 saat ia bekerja di CERN (European Organization for Nuclear Research) di Swiss. Waktu itu, Tim ingin menemukan cara agar para peneliti di seluruh dunia bisa berbagi dokumen dan informasi dengan mudah melalui jaringan komputer yang akhirnya menjadi cikal bakal World Wide Web (WWW) üåç Versi pertama HTML masih sangat sederhana, hanya punya sedikit tag seperti :

<head>
<title>Contoh Halaman</title>
</head>
<body>
<p>Halaman web pertama di dunia!</p>
</body>
</html>
  • 1993-HTML 1.0 dirilis secara resmi ke publik.
  • 1995-HTML 2.0 hadir dengan lebih banyak tag seperti <form> untuk formulir
  • 1997-HTML 3.2 memperkenalkan tabel, font, dan elemen presentasi.
  • 1999-HTML 4.01 jadi standar yang banyak dipakai di awal 2000-an.
  • 2014-HTML5 dirilis dan membawa bayak pembaruan seperti video, audio, canvas, dan struktur semantik (<header>, <footer>, <articel>).

Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat halaman web. HTML bekerja dengan cara memberikan markup atau tanda pada teks agar browser tahu bagaimana cara menampilkannya. Misalnya, teks yang dikelilingi tag <h1> akan muncul sebagai judul besar, sedangkan teks dalam tag <p> akan tampil sebagai paragraf.

HTML bukan bahasa pemrograman seperti Python atau JavaScript, tapi bahasa markup artinya HTML berfungsi untuk mengatur struktur dan isi halaman web, bukan untuk menjalankan logika program.

Bagaimana Cara Kerja HTML?

  • Kamu dapat meenulis kode HTML dalam file berekstensi .html
  • Browser seperti Chrome atau Firefox membaca kode tersebut
  • Browser menampilkan hasilnya di layar sesuai struktur

Fungsi Utama HTML

    Fungsi-fungsi utama HTML antara lain :

  • Menyusun Struktur Halaman Web : HTML menentukan bagian mana yang menjadi judul, paragraf, gambar, tautan, tabel, dan elemen lainnya. dengan HTML, halaman wen jadi punya bentuk yang teratur. HTML menentukan bagian mana yang jadi header, konten utama, sidebar, footer, dan sebagainya. Bayangin HTML itu seperti kerangka bangunan rumah, sebelum dicat dan dihias oleh CSS.
  • Menampilkan Konten ke Browser : HTML menampilkan teks, gambar, audio, video, dan tautan agar bisa dilihat pengguna di browser.
  • Menghubungkan Antarhalaman : dengan tag <a>, HTML bisa menghubungkan satu halaman ke halaman lain. Ini dasar dari sistem navigasi web. HTML memungkinkan setiap halaman saling terhubung ‚Äî itulah sebabnya internet disebut World Wide Web, yaitu ‚Äújaringan halaman yang saling terhubung.‚Äù
  • Sebagai Dasar untuk CSS dan JavaScript : HTML adalah pondasi dari website. HTML menyusun konten, CSS mempercantik tampilannya, dan JavaScript menambahkan interaksi. Ketiganya saling melengkapi tetapi HTML selalu jadi dasar utamanya.
  • Membantu Mesin Pencari (SEO) : Struktur HTML yang tapi memudahkan mesin pencari seperti Google memahami isi halamanmu. Semua browser modern (Chrome, Edge, Firefox, Safari, Opera) menggunakan HTML untuk membaca dan menampilkan halaman web. Artinya, selama kamu bisa menulis HTML, karyamu bisa diakses siapa saja di seluruh dunia.
  • Dasar dari Pengembangan Web Modern : Semua website dari Google, YouTube, sampai web yang kamu buat, dibangun di atas HTML. Tanpa HTML, browser nggak akan tahu apa yang harus ditampilkan ke pengguna. Mau belajar frontend, backend, desain UI/UX, atau full stack , semuanya dimulai dari HTML.
